### Project execution environment: Node.js v20.14.0
https://nodejs.org/download/release/v20.14.0/

How to execute (in development mode):
1. clone code
2. install library: npm i
3. Update file .env.development (optional)
4. Run the project: npm run dev

=============

How to execute (in production mode):
1. clone code
2. install library: npm i
3. Update file .env.production (optional)
4. Build dự án: npm run build
5. Run the project dự án: npm run preview
